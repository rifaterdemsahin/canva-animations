üß† The Concept

Environment variables are values that live outside of an application's code, in the environment where the app runs. They are used to configure an application, for example, to store sensitive data like API keys instead of writing them directly in the code.

Follow these clicks to create the animation in Canva:

  • Step 1: Setup. Create a new "Video" design.
  • Step 2: Add Elements. In "Elements", add a "file" icon and a "docker" or "container" icon. Add text inside the container like `[API_KEY]`.
  • Step 3: Animate Injection.
    • Add a text element next to the file icon with the secret value (e.g., `xyz123`).
    • Select this text element, click "Animate" > "Drift", and set its path to move over the placeholder text in the container.
  • Step 4: Animate Replacement.
    • Select the placeholder text. Click "Animate" > "Fade" and set it to "On Exit".
    • Select the secret value text again. Add a second animation: "Fade" set to "On Enter".
  • Step 5: Adjust Timing. In the timeline, make the secret value's "Drift" happen first. Then, have the placeholder's "Fade" out and the secret's "Fade" in happen simultaneously.
