Concept: CSS Box Model

üß† The Concept

In web design, every HTML element is a rectangular box. The CSS Box Model describes the layers of this box: the `content` area, surrounded by `padding`, a `border`, and finally an outer `margin`.

            graph TD
                M("Margin") --> B("Border")
                B --> P("Padding")
                P --> C["Content"]
